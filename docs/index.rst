Dataset Comparison Tool
======================

A comprehensive tool for dataset comparison, validation, and analysis.

Quick Start
----------

.. code-block:: bash

   pip install -r requirements.txt
   streamlit run app.py

Feature Overview
--------------

* Interactive file upload and comparison
* Customizable mapping rules
* Data validation and quality checks
* Large dataset support with Dask
* Visualization and reporting

For detailed information, please refer to the respective sections in the documentation.
